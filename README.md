# ElectronicUntis
An alternative mobile client for the Untis timetable system with a focus on simplicity and Electronic Design.

## Top features
- Modern and intuitive Electronic Design
- Simple and straight-forward login process
- Fast timetable loading
- Focus on Timetable and Rooms
- Break notifications and auto-mute during lessons

## Development
This is a modified version of the ElectronicUntis project, streamlined for specific use cases.
